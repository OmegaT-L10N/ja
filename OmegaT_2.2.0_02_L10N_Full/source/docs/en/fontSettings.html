Font settings

This dialog allows you to modify the font used by OmegaT in the following windows:

  • OmegaT main window (Editor, Match viewer, Glossary viewer)

  • Search window

  • Tag validation window

The dialog can be called up via the Options â†’ Font... item in the Main menu.

The dialog allows you to adjust the font settings as follows:

  • Use the drop-down box to select among the fonts available

  • Use the spin box to adjust size of the selected font

  • The sample text in the preview frame is shown using the selected font, so you can check it immediately

Warning! In some cases it make take quite some time for OmegaT to update the display after you have changed the font setting, especially when a big file with many segments is open in the editor. Note also that some fonts behave better for some language pairs than for others.
